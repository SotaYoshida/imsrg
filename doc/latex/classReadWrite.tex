This means that the model space of the file must also be specified. emax refers to the maximum single-\/particle oscillator shell. Emax refers to the maximum of the sum of two single particles. lmax refers to the maximum single-\/particle $ \ell $. If Emax is not specified, it should be 2 $\times$ emax.
